#/`mak`

`make`, but `mak` it faster.

Run `make` with maximum parallelism and get a live overview of the progress.

## Install

```shell
brew install --HEAD lgarron/lgarron/mak
```

## Examples

```shell
git clone https://github.com/cubing/cubing.js && cd cubing.js
npm install

mak test-all
```
